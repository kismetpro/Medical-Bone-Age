## 📋 技术细节

### 后端处理流程
```
1. 接收前端请求 (use_dpv3=true)
   ↓
2. 调用 DPV3BoneDetector.detect()
   ↓
3. YOLO检测21个标准骨骼
   ↓
4. 手指分类（按RUS-CHN标准）
   ↓
5. 格式化关节数据
   ↓
6. 返回JSON给前端
```

### 前端渲染流程
```
1. 接收JSON数据
   ↓
2. 提取 joint_detect_13
   ↓
3. 显示 plot_image_base64 (如果有)
   ↓
4. 显示 detected_count 和 hand_side
   ↓
5. 渲染 chartData (分级分布图)
   ↓
6. 渲染 recognizedRows (明细表)
```

## 🐛 故障排查

### 查看后端日志
```bash
# 启动后端时观察日志
python entry_point.py

# 期望看到:
✅ YOLO模型加载成功
✅ DP V3 bone detector loaded successfully
🚀 服务器已启动: http://0.0.0.0:8000
```

### 查看前端网络请求
1. 打开浏览器开发者工具 (F12)
2. 选择 Network 标签
3. 上传图片并提交
4. 查找 `joint-grading` 请求
5. 检查:
   - Status: 200 (成功)
   - Response: 包含 `joint_detect_13`
   - `dpv3_enhanced`: true

### 常见错误

#### "DP V3检测器未加载"
**原因**: DPV3BoneDetector初始化失败
**解决**:
```bash
cd backend
python -c "from dp_bone_detector_v3 import DPV3BoneDetector; d = DPV3BoneDetector()"
```

#### "小关节分级模型未加载"
**原因**: joint_grader未初始化
**解决**: 检查 `app/models/joint` 目录下的模型文件

#### 前端显示空白
**原因**: Response数据格式不正确
**解决**: 打开浏览器控制台查看具体错误
